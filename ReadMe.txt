The source code Folder contains the following:
Provided data:  TRAINING WINDOW INDEXES.dat
	        RED BAND.TIF
		GREEN BAND.TIF
		BLUE BAND.TIF
		INFRARED BAND.TIF

Source code files: merge.py
		   segmentation.py
		   acuracy.py

Legend image:	legend.jpg

Classified output image: CLASSIFIED IMAGE.TIF

1. Run the merge.py script first to obtain required band combinations(GRI.TIF,     RIB.TIF, IGB.TIF)
2. Run the segmentation.py script next to obtain the classified output image    (CLASSIFIED IMAGE.TIF)
3. Run the accuracy.py script in the last to obtain algorithm acuuracy in    accuracy.txt file
